# Galvanostatic Intermittent Titration Technique on graphite with 5% silicon

This repository contains a subset of the data published in the research paper:
**D. Wycisk et al., "Challenges of open-circuit voltage measurements for silicon-containing Li-Ion cells,"** *Journal of Energy Storage*, vol. 89, p. 111617, June 2024. [DOI: 10.1016/j.est.2024.111617](https://doi.org/10.1016/j.est.2024.111617)
